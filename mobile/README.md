# Mobile shells (`mobile/`)

Native iOS / Android shells around the PWA, generated by
[Capacitor](https://capacitorjs.com/). The PWA bundle in `build/` is
copied into `mobile/webdir/` at build time; the Capacitor projects load
that as their webview.

## Layout

| Path | Owns |
|---|---|
| `ios/` | Xcode project. Open `ios/App/App.xcworkspace` in Xcode to build/sign. |
| `android/` | Gradle project. Open in Android Studio or run `./gradlew assembleDebug`. |
| `webdir/` | PWA bundle copy target (gitignored — populated by `npx cap copy`). |

## Build

```bash
npm run build                    # produce build/*.mjs
npx cap copy                     # sync build/ → mobile/webdir/
npx cap open ios                 # open Xcode
npx cap open android             # open Android Studio
```

See [`docs/MAC_BOOTSTRAP.md`](../docs/MAC_BOOTSTRAP.md) for Xcode +

## Why Capacitor over Cordova / native

PWA-first stays the source of truth for the chat surface; Capacitor
gives us native lockscreen audio, mic-while-locked, push, and IAP
without forking the codebase.

See the top-level [`README.md`](../README.md) for the broader
architecture.
